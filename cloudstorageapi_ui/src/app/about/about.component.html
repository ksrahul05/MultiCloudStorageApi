This project is a web application that offers a cohesive and user-friendly experience for managing files across Google Drive and OneDrive. It simplifies file management by allowing users to upload files to both platforms simultaneously, transfer files between them, and download files as needed. Provide a centralized platform for managing files across Google Drive and OneDrive. Enhance file management efficiency and user productivity. Ensure secure and seamless interaction with cloud APIs using OAuth2 and JWT.

User Guide

Setup:

Access the application URL.
For the first time create an account by providing your details in this platform.
Log in using your Google or Microsoft account credentials.

Using the Application:

Upload files by selecting both platforms.
Drag and Drop feature available to upload files.
Use the transfer feature to move files between Google Drive and OneDrive.
Download files by navigating to the desired storage service.

Troubleshooting:

Issue: Cannot log in.
Solution: Verify API key configuration and network connectivity.
